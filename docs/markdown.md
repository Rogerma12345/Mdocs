# 5分钟了解Markdown

## 标题
```
# H1 标题
## H2 标题
### H3 标题
#### H4 标题
##### H5 标题
###### H6 标题
```

## 文本格式
```
**粗体文本**
*斜体文本*
***粗体和斜体***
~~删除线~~
`行内代码`
```

## 链接和图片
```
[链接文本](https://example.com)
[带标题的链接](https://example.com "悬停标题")
![替代文本](image.jpg)![带标题的图片](image.jpg "图片标题")
```

## 列表
```
无序列表:
- 列表项 1
- 列表项 2
  - 嵌套列表项

有序列表:
1. 第一项
2. 第二项
3. 第三项
```

## 引用块
```
> 这是一个引用块
> 多行引用
>> 嵌套引用
```

## 代码块
````
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## 表格
```
| 表头 1 | 表头 2 | 表头 3 |
|----------|----------|----------|
| 行 1    | 数据     | 数据     |
| 行 2    | 数据     | 数据     |
```

## 水平线
```
---
或
***
或
___
```

## 任务列表
```
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务
```

## 字符转义
```
使用反斜杠进行转义: \* \_ \# \`
```

## 换行
```
在一行末尾使用两个空格  
来创建一个换行符。

或者使用一个空行来开始一个新段落。
